# Wild Oasis

## Overview

**Wild Oasis** is an internal web application designed to streamline the management of user check-ins, check-outs, and other administrative tasks. Built with React and Vite, it offers a responsive and efficient interface for staff members to handle daily operations effectively.

## Features

- **User Management**: Register new users, update user information, and manage user statuses.
- **Check-In/Check-Out System**: Efficiently log user check-ins and check-outs with timestamps.
- **Dashboard**: Overview of current user statuses, recent activities, and key metrics.
- **Search Functionality**: Quickly locate user records using the integrated search feature.
- **Responsive Design**: Optimized for various devices, ensuring accessibility for all staff members.

## Technologies Used

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: CSS Modules
- **State Management**: Context API
- **Linting**: ESLint

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/wild-oasis.git
   cd wild-oasis
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Set Up Environment Variables**:

   Create a `.env` file in the root directory and add any necessary environment variables, such as API endpoints or authentication keys.

4. **Start the Development Server**:

   ```bash
   npm run dev
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **User Authentication**: Staff members can log in using their credentials to access the system.
2. **Dashboard Navigation**: Upon logging in, the dashboard provides an overview of user activities and system metrics.
3. **Managing Users**: Navigate to the user management section to add new users, edit existing user information, or change user statuses.
4. **Logging Check-Ins/Check-Outs**: Use the check-in/check-out interface to record user entries and exits efficiently.
